# Minecraft-teamplan

## 目標

サバイバルモード(ゲーム内で敵や体力がありゲームオーバーがあるモードのこと)でITカレッジを制作し、そこから近辺の建物を再現する。  



## 活動内容
### 5/19

- GoogleCloudPlatform（GCP）の無料期間を使用して一時的にサーバーを建てる
- この無料期間内で借りるサーバーの種類やかかる料金など調べながら活動する

### 6/15
